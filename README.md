## BeatBox Converter

Turn your vocal perc into actual sample beats.
Human Beatboxing --> Sound Production

--

## Get Started

1. **Install dependencies**

```bash
npm install
```

2. **Run dev servers**

```bash
npm run dev
```

3. **View App**
   Open [http://localhost:3000](http://localhost:3000) with your browser.

## Learn More

- Framework: Next.js
- Language: TypeScript
- Styling: Tailwind
